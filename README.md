# tunebooth_client
This is the client app side of tunebooth built in React Native
# To run the project, follow these steps:
1. Pull the project in your system.
2. Run npm install command in the root directory of the project
3. Configure the android, ios or expo environment according to your system(Full guide - https://reactnative.dev/docs/environment-setup)
4. After setting up the development environment, run the project with suitable command given in the documentation mentioned above.
